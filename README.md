#Phần Mềm Quản Lý Sinh Viên

Đây là ứng dụng Windows Forms đơn giản được xây dựng bằng C# để hỗ trợ việc quản lý thông tin sinh viên, điểm số và lớp học một cách hiệu quả và trực quan.

##Chức Năng
Thêm mới: Hỗ trợ nhập liệu nhanh, tự động nhảy dòng (Enter).
Cập nhật: Sửa thông tin sinh viên dễ dàng.
Xóa: Xóa sinh viên với hộp thoại xác nhận an toàn.
Tìm kiếm & Sắp xếp: Tìm kiếm kết hợp nhiều điều kiện / Sắp xếp danh sách theo Điểm (Tăng/Giảm) hoặc Tên (A-Z/Z-A theo chuẩn tên tiếng Việt).
Lưu & Mở file: Hỗ trợ lưu trữ dữ liệu ra file .txt để sử dụng lâu dài.


###Công Nghệ Sử Dụng
